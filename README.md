### Featured Projects

**🔌 Smart Universal Programmable Charger**
Designed the control logic for a smart battery charger supporting Constant Voltage (CV) and Constant Current (CC) modes, simulating real sensor-feedback behavior in C++. The program compares target vs. measured voltage/current and decides whether to increase, decrease, or hold output — the same decision logic a microcontroller would run in a real charger. Also mapped out the full hardware design for a physical build (Arduino Nano, INA219 voltage/current sensor, DPS5005 programmable buck converter, LCD display, and protection circuitry) and built a GUI to visualize charging status in real time.
`C++` `Embedded Systems Logic` `Feedback Control` `Arduino` `Power Electronics`

**⚡ Maxwell Inductance Bridge**
Designed, simulated, and physically built a Maxwell Inductance Bridge circuit to measure an unknown inductance by balancing it against a known standard inductor. Derived the balance equations from first principles, calculated theoretical values, and used variable resistors to fine-tune the bridge until the detector voltage approached zero — accounting for real-world component tolerances. Validated the design in Proteus simulation and on a physical breadboard, achieving a measured inductance within close range of the expected 330 µH.
`Circuit Analysis` `AC Bridge Networks` `Proteus Simulation` `Hardware Prototyping`

**💡 LED Intensity Control Circuit**
Designed and built a circuit to control the brightness of an LED by varying the current/voltage delivered to it, exploring practical dimming techniques relevant to power electronics and embedded control. This project laid groundwork for later work with PWM-based control — a core concept in motor drivers, EV battery systems, and embedded lighting applications.
`Circuit Design` `Analog Electronics` `PWM Fundamentals`
